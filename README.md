#  QuizTech

Projeto desenvolvido para praticar HTML, CSS e JavaScript.

##  Sobre

O QuizTech é um quiz de programação onde o usuário responde perguntas de múltipla escolha e recebe um feedback imediato indicando se acertou ou errou.

Ao final do quiz é apresentada a pontuação juntamente com uma mensagem de desempenho.

##  Funcionalidades

- Perguntas carregadas por JSON
- Feedback imediato
- Pontuação final
- Tema Claro/Escuro
- Design responsivo
- Animação entre perguntas

##  Tecnologias

- HTML5
- CSS3
- JavaScript
- JSON
